# Seleção automatica de imagens

> The main asset is assumed to correspond to a resolution of 1.0. For example, consider the following asset layout for an image named my_icon.png:

```
content_copy
  .../my_icon.png
  .../2.0x/my_icon.png
  .../3.0x/my_icon.png
```
