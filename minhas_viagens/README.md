# Minhas Viagens

App simples utilizando GoogleMaps

![Minhas Viagens logo](../asstes/images/minhas-viagens.jpg)

## Recursos utilizados
 * [Google Maps for Flutter (Developers Preview)](https://pub.dev/packages/google_maps_flutter)
 * [Flutter Geolocator Plugin](https://pub.dev/packages/geolocator)
 * [Multidex](https://developer.android.com/studio/build/multidex)
 * [Firebase Core for Flutter](https://pub.dev/packages/firebase_core)
 * [Cloud Firestore Plugin for Flutter](https://pub.dev/packages/cloud_firestore)
